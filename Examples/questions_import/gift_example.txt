::GIFT Control Characters::<BR>
Which of the following is NOT a control character for the GIFT import format? {<BR>
   ~ \~     # \~ is a control character.<BR>
   ~ \=     # \= is a control character.<BR>
   ~ \#     # \# is a control character.<BR>
   ~ \{     # \{ is a control character.<BR>
   ~ \}     # \} is a control character.<BR>
   = \\     # Correct! \\ (backslash) is not a control character. BUT,<BR>
              it is used to escape the control characters. So, to specify<BR>
              a literal backslash, you must escape it with a backslash<BR>
              (as shown in this example).<BR>
}<BR>
